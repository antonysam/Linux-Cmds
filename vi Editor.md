# vi Editor

_Types of mode_
```
command (ESC) 
insert (i)
```
_Navigation commands_
```
left  --> press h
right --> press l

move up   --> press j
move down --> press k
```
_Delete, Copy & Paste_

```
dd        --> delete lines
yy (yank) --> copy lines
p         --> paste lines
```
_Saving & quitting_
```
:wq
or
:wq!
```
